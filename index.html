Desarrollado por @manuelmsni

Este programa cifra textos en función de una contraseña (obligatoria) y una semilla (opcional), no podrás descifrar el texto resultante si no tienes la contraseña y la semilla con las cuales fueron originados.

Este programa calcula el código hash de un texto basandose en el algoritmo Sha3 importado desde el repositorio en github de Chris Veness.
